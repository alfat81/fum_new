# Сайт Комиссии по фигурному управлению мотоциклом (мотоджимхане) МФР

## Общая информация
Современный адаптивный сайт для Комиссии по фигурному управлению мотоциклом (мотоджимхане) при Мотоциклетной Федерации России. Сайт полностью переработан для современного вида и удобства управления.

## Структура файлов

### Основные файлы
- `index.html` - Главная страница
- `about.html` - О комиссии
- `structure.html` - Структура комиссии
- `calendar.html` - Календарь соревнований
- `documents.html` - Полезные документы
- `contacts.html` - Контакты
- `partners.html` - Партнеры

### Общие компоненты
- `header.html` - Шапка сайта (выделена в отдельный файл для удобства редактирования)
- `footer.html` - Подвал сайта (выделен в отдельный файл для удобства редактирования)
- `admin-panel.html` - Админ-панель (выделена в отдельный файл для удобства редактирования)

### JavaScript
- `main.js` - Основной JavaScript файл со всей логикой сайта, включая:
  - Мобильное меню
  - Админ-панель
  - Управление структурой комиссии
  - Управление региональными представителями
  - Настройки страниц
  - Анимации и интерактивные элементы

### CSS
- `styles.css` - Основной CSS файл со всеми стилями сайта, включая:
  - Адаптивную верстку
  - Современные анимации
  - Стили для админ-панели
  - Медиа-запросы для всех устройств

## Особенности для GitHub Pages

### Преимущества текущей структуры
1. **Модульность** - Шапка и подвал выделены в отдельные файлы, что позволяет легко обновлять их во всех страницах
2. **Легкость редактирования** - Все страницы имеют одинаковую структуру, изменения вносятся быстро
3. **Отсутствие серверной части** - Сайт полностью статический, идеально подходит для GitHub Pages
4. **Админ-панель без бэкенда** - Вся логика управления данными реализована на JavaScript с сохранением в localStorage
5. **Современный дизайн** - Адаптивный дизайн с анимациями и интерактивными элементами

### Рекомендации по дальнейшей работе

#### 1. Для обновления шапки/подвала
- Откройте файл `header.html` или `footer.html`
- Внесите изменения
- Замените соответствующие блоки во всех HTML файлах (можно использовать поиск/замену по коду)

#### 2. Для добавления новых страниц
- Скопируйте любой существующий HTML файл
- Замените контент в `<main class="content">`
- Обновите navigation links в шапке
- Добавьте файл в корень репозитория

#### 3. Для управления контентом через админ-панель
- По умолчанию пароль: `fum2025admin` (рекомендуется изменить в `main.js`)
- Все изменения сохраняются в localStorage браузера
- Для экспорта данных используйте кнопку "Экспорт данных" в админ-панели
- Для импорта данных используйте файл формата JSON

#### 4. Для оптимизации под GitHub Pages
- Все пути к файлам относительные (`./images/`, `./styles.css`)
- Нет зависимостей от внешних серверов
- Минимальное количество HTTP запросов
- Оптимизированные изображения (в папке `images/`)

#### 5. Для SEO оптимизации
- Каждая страница имеет уникальный title и meta description
- Семантическая разметка HTML5
- Быстрая загрузка страниц
- Адаптивный дизайн для мобильных устройств
